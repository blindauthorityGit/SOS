== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Jupiter X in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://help.artbees.net/ on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.

== How to create a page like theme screenshot by using Gutenberg ==

1. Create a new page.
2. Set Page Attributes > Template to Full Width.
3. Add a Cover block.
4. Add some texts and button in cover.
5. Set a proper color and alignment for texts and button.
6. Add a Column Block and set columns number to 5.
7. For each column, add an image and proper caption.
8. Save your page.
Enjoy it!

* Shop basket icon will be available after installing WooCommerce plugin.
* Language icon will be available after installing multilingual plugins like Polylang.
* For more customization options, please install and activate Jupiter X Core plugin.

== How to create a page like theme screenshot by using Elementor ==
1. Go to WP dashboard > Add new page. From the right dashboard set the layout to Elementor Full Width. Update. Click on Edit with Elementor blue button. Elementor with a left dashboard is activated now.
2. From Elementor Dashboard > add your elements
3. When you’re done, SAVE your page and get a Preview
4. From top bar of same Preview go to this link > Customize. Customizer is now activated.
5. Customizer > Site Identity: add your logo
6. Customizer > Header > Settings tab Arrange your Display Elements like Logo, Search, Menu and Cart. As you see, the Header Type is automatically set to Default. For further premade header templates, you can choose Custom tab.
7. Customizer > Header > Styles tab For details styling of Display Elements, you can use this tan.
8. Click on Publish button.
* Video tutorial: https://www.dropbox.com/s/jevmt4mz52ow5nn/Sample.mp4
Enjoy it!
